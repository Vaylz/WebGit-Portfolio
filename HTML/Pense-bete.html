Commande Linux


Navigation dans le système de fichiers :


  • ls : Liste les fichiers et répertoires.
  • cd : Change le répertoire courant.
  • pwd : Affiche le chemin du répertoire courant.

    • Manipulation de fichiers et répertoires :


  • cp : Copie des fichiers ou des répertoires.
  • mv : Déplace ou renomme des fichiers ou des répertoires.
  • rm : Supprime des fichiers ou des répertoires.
  • mkdir : Crée un nouveau répertoire.

    • Affichage de contenu :


  • cat : Copie des fichiers ou des répertoires.
  • more et less : Affichent le contenu d'un fichier page par page.
  • head et tail : Affichent les premières ou dernières lignes d'un fichier.

    • Édition de fichiers texte :


  • nano : Éditeur de texte simple en ligne de commande.
  • vim et emacs : Éditeurs de texte plus avancés.

    • Gestion des processus :


  • ps : Affiche les processus en cours d'exécution.
  • kill: Tue un processus en utilisant son PID.
  • top : Affiche les processus en cours d'exécution de manière interactive.

    • Gestion des utilisateurs et des permissions :


  • useradd et userdel : Ajoutent et suppriment des utilisateurs.
  • passwd : Modifier le mot de passe d'un utilisateur.
  • chmod : Modifie les permissions d'un fichier ou d'un répertoire.

    • Recherche de fichiers et de texte :


  • find : Recherche des fichiers dans un répertoire donné.
  • grep : Recherche des expressions régulières dans des fichiers.

    • Réseau :


  • ping : Vérifie la connectivité réseau.
  • ifconfig et ip : Affichent et configurent les interfaces réseau.
  • nslookup et dig : Effectuent des requêtes DNS.

    • Archivage et compression :


  • tar : Archive des fichiers et répertoires.
  • gzip et gunzip : Compression et décompression au format gzip.
  • zip et unzip : Compression et décompression au format zip.

    • Autres :


  • echo : Affiche du texte à la sortie standard.
  • date : Affiche la date et l'heure actuelles.
  • man : Affiche le manuel d'utilisation d'une commande.

    Commande SQL


    Création de tables :


  • CREATE TABLE : Crée une nouvelle table dans la base de données.

    • CREATE TABLE nom_de_table ( colonne1_type contraintes, colonne2_type contraintes, ... );

    Insertion de données :


  • INSERT INTO : Insère des données dans une table.

    • INSERT INTO nom_de_table (colonne1, colonne2, ...)
    VALUES (valeur1, valeur2, ...);

    Sélection de données :


  • SELECT :s Récupère des données depuis une ou plusieurs tables.

    • SELECT colonne1, colonne2, ... FROM nom_de_table WHERE condition;

    Mise à jour de données :


  • UPDATE : Modifie les données existantes dans une table.

    • UPDATE nom_de_table
    SET colonne1 = nouvelle_valeur1, colonne2 = nouvelle_valeur2, ... WHERE condition;

    Suppression de données :


  • DELETE FROM : Supprime des données d'une table.

    • DELETE FROM nom_de_table WHERE condition;

    Filtrage et tri des résultats :


  • WHERE : Filtre les résultats basés sur une condition.
  • ORDER BY :Trie les résultats par une ou plusieurs colonnes.

    • SELECT colonne1, colonne2, ...
    FROM nom_de_table
    WHERE condition
    ORDER BY colonne1 ASC/DESC, colonne2 ASC/DESC;

    Agrégation des données :


  • GROUP BY : Groupe les résultats par une ou plusieurs colonnes.
  • SUM, AVG, COUNT, MAX, MIN : Fonctions d'agrégation.

    • SELECT colonne1,COUNT(colonne2) AS nombre
    FROM nom_de_table GROUP BY colonne1;

    Jointures de tables :


  • INNER JOIN, LEFT JOIN, RIGHT JOIN : Combine les lignes de plusieurs tables basé sur une condition.

    • SELECT t1.colonne1, t2.colonne2
    FROM table1 t1
    INNER JOIN table2 t2 ON t1.id = t2.id;

    Index :


  • CREATE INDEX : Crée un index sur une colonne pour améliorer les performances de recherche.

    • CREATE INDEX index_nom ON nom_de_table (colonne);

    Transactions :


  • BEGIN TRANSACTION, COMMIT, ROLLBACK : Gestion des transactions.

    • BEGIN TRANSACTION; -- des opérations SQL
    COMMIT; -- ou ROLLBACK en cas d'annulation
